# Roistat Tracking Link Demo

Небольшое frontend-приложение для подготовки рекламных ссылок с UTM- и Roistat-метками.

Проект имитирует работу рядом с Roistat: можно создать demo-проект, собрать ссылку, проверить предупреждения, вести кампании, историю и экспортировать проект в JSON.

## Быстрый запуск

Нужен Node.js 20+ и npm.

### Windows

```powershell
git clone https://github.com/<your-username>/roistat-tracking-link-test.git
cd roistat-tracking-link-test
npm install
npm run dev
```

Откройте в браузере:

```text
http://127.0.0.1:5173/start
```

### macOS / Linux

```bash
git clone https://github.com/<your-username>/roistat-tracking-link-test.git
cd roistat-tracking-link-test
npm install
npm run dev
```

Откройте:

```text
http://127.0.0.1:5173/start
```

## Что умеет приложение

- Создавать demo-проект с Project ID, доменом, источниками и бюджетом.
- Генерировать рекламные ссылки с `utm_*`, `roistat` и `roistat_param1..5`.
- Использовать Roistat marker разрешенного источника при генерации `roistat`.
- Показывать предупреждения перед созданием рискованной ссылки.
- Автоматически добавлять новую кампанию после успешного создания ссылки.
- Контролировать бюджет проекта и кампаний.
- Вести историю действий: созданные ссылки, кампании, настройки, ошибки.
- Экспортировать активный проект в JSON и импортировать его как новый проект.
- Показывать QR-код для готовой ссылки.

## Как пользоваться

1. На стартовом экране нажмите `Создать demo-проект`.
2. Перейдите в `Создать ссылку`.
3. Заполните URL, `utm_source`, `utm_campaign` и бюджет.
4. Если появились предупреждения, проверьте их и подтвердите только если данные корректны.
5. После создания ссылка появится на главной странице, а новая кампания попадет в раздел `Кампании`.

## Экспорт и импорт

Экспорт JSON находится в разделе `Настройки`.

Экспортируется только активный проект: его настройки, кампании и история.

Если в JSON изменить `Project ID` на уникальный, импорт добавит проект к уже существующим. Если `Project ID` уже есть в приложении, импорт будет остановлен с ошибкой.
